# site_test

Minimal clothing storefront scaffold built for the `gwon1023/site_test` repository.

## Included

- Home page with a compressed structure inspired by the reference store
- Product catalog
- Product detail with size, color, quantity, shipping, and returns
- Cart persisted in `localStorage`
- Checkout page with order summary
- Simulated order completion flow
- Agent prompt planning document for the project workflow

## Commands

```bash
npm install
npm run dev
```
